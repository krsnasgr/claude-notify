# Changelog

## [0.1.0] - 2026-03-14

### Added

- Native OS notifications when Claude Code finishes a task
- Notification when Claude requires user input
- Notification when Claude requires permission to proceed
- Status bar toggle to enable/disable notifications
- Test notification command
- Configurable cooldown between notifications
- Optional sound with notifications (macOS)
- Cross-platform support: macOS, Linux, Windows
- Automatic hook installation/removal in Claude Code settings
